## Usage
```
python main.py
```
In game display

space key: pause
esc: exit game
right arrow:  speed up
left arrow: speed down

push number button and lord pre life
0 or delete key: delete all cell
1: gulider gun
2: donguri
3: die hard
4: horizontal line
5: vertical line
9: random
